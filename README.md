# Simulador de Navegador em Python

Projeto acadêmico desenvolvido para praticar Programação Orientada a Objetos e Estrutura de Dados.

O sistema simula um navegador no terminal, utilizando uma pilha para controlar o histórico de navegação.

## Funcionalidades
- Acessar URLs
- Visualizar conteúdo das páginas
- Navegar entre links internos
- Voltar para a página anterior (#back)
- Visualizar histórico completo (#showhist)
- Adicionar novos sites (#add)

## Conceitos aplicados
- Programação Orientada a Objetos
- Encapsulamento
- Estrutura de Dados (Pilha)
- Tratamento de Exceções
- Leitura de Arquivos
